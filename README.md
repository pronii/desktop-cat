# desktop-cat

Windows desktop pet prototype built with Electron.

## First version scope

- Transparent frameless floating window
- Always-on-top display by default
- Draggable Q-style cat
- Idle breathing, blinking, and tail animation
- Small random roaming movement
- Happy feedback when clicked
- Right-click menu with Quit

Not included yet: feeding, mood, levels, settings, external image assets, or an installer.

## Run locally

```powershell
npm install
npm start
```

## Test

```powershell
npm test
```
